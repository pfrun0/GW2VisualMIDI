# GW2VisualMIDI

A Windows app that lets you play Guild Wars 2 instruments using a MIDI piano keyboard. Translates MIDI input to keyboard commands with network-adaptive timing.

## Download

You can clone the repo and run `GW2VisualMIDI.sln` with visual studio and build it yourself or just head to the latest release for the binaries

## Requirements

- MIDI keyboard/controller

## Quick Start

1. Plug in your MIDI keyboard
2. Launch Guild Wars 2
3. Run `GW2VisualMIDI.exe`
4. Start playing!

## Features

- Real-time MIDI to keyboard translation
- Visual piano display with key feedback
- Network-adaptive octave shift delays
- Fully rebindable keys
